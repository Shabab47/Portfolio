# Tawfik Rahman Shabab — Portfolio

A modern, responsive personal portfolio website showcasing my projects, skills, and experience as a Computer Science & Engineering student at BAUET.

[Live Demo](https://shabab47.github.io/Portfolio/)

---

## Features

- Clean dark theme with accent gradients and animations
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation with active section highlighting
- Scroll-triggered reveal animations
- Project showcase with links to live demos and source code
- Contact form powered by EmailJS
- Interactive education timeline

## Built With

- **HTML5** — semantic structure
- **CSS3** — custom properties, CSS Grid, Flexbox, animations
- **Vanilla JavaScript** — Intersection Observer, DOM manipulation
- **EmailJS** — serverless contact form delivery
- **GitHub Pages** — hosting and deployment

## Project Structure

```
Portfolio/
├── index.html        # Page structure and content
├── css/
│   └── style.css     # All styling
├── js/
│   └── main.js       # All interactivity
├── profile.jpeg
└── README.md
```

## Run Locally

```bash
cd Portfolio
python -m http.server 5500
# or: npx serve
```

Open http://localhost:5500 in your browser.
